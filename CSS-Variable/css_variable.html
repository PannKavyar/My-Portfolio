ဖြစ်ဖို့ရန်အတွက် CSS variables တွေကို အသုံးပြုသွားမှာဖြစ်ပါတယ်။ Variables တွေဟာ programming မှာ တစ်ကယ်ကို အရေးအကြီးဆုံး concepts တွေထဲက တစ်ခုဖြစ်ပါတယ်။ သူတို့ဟာ information တွေကို track လုပ်ဖို့နဲ့ ဖြစ်နိုင်တဲ့ values တွေကိုပြန်ပြီး ကိုးကားထားတာဖြစ်ပါတယ်။ CSS variables တွေကို formerly အရ Custom Properties တွေလို့ သိကြပါတယ်။ ဘာကြောင့်လဲဆိုရင် သူတို့ဟာ regular CSS properties တွေနဲ့ဆင်တူပြီး custom Declaring and Using CSS Variables

CSS variables တွေကို ကျွန်တော်တို့ style sheet ထဲမှာသုံးဖို့အတွက် step process နှစ်ခုရှိပါတယ်။ ပထမတစ်ခုက ကျွန်တော်တို့ဟာ custom property notation ကိုသုံးပြီး selector တစ်ခုအတွင်းမှာ variable ကို declare လုပ်ရမှာဖြစ်ပါတယ်။ ထို့နောက် ကျွန်တော်တို့က အဲ့ဒီ variable or custom property name ကို var( ) function ကိုသုံးပြီး reference လုပ်ပေးရမှာဖြစ်ပါတယ်။ CSS variables တွေဟာ regular CSS properties တွေလိုပဲ cascade ရဲ့ rules တွေအတိုင်း လိုက်နာတဲ့အတွက် သူတို့ကို rule set ကိုအတွင်းက element တစ်ခုပေါ်မှာ ပထမဆုံး declare လုပ်ရပါမယ်။ တစ်ကယ်လို့ rule တစ်ခုရဲ့ အပြင်ဘက်မှာ declare လုပ်ခဲ့မယ်ဆိုရင် ကျွန်တော်တို့ style sheet အတွင်းမှာ variable ကို အသုံးပြုလို့ရမှာမဟုတ်သလို အလုပ်လုပ်မှာလည်း မဟုတ်ပါဘူး။ ပြီးတော့ ကျွန်တော်တို့ဟာ main brand color, base font-size လိုမျိုး elements အများကြီးကိုရည်ညွှန်းတဲ့ variables ကို မကြာခဏဆိုသလို :root pseudo class ကိုသုံးပြီး root element ပေါ်မှာ declare လုပ်ရပါမယ်။ :root pseudo class ဟာ page ပေါ်မှာ (သို့) highest level parent ပေါ်မှာရှိတဲ့ root element ကို target လုပ်ပါတယ်။ ဥပမာ ကျွန်တော်တို့ HTML file ထဲမှာဆိုရင် ဟာ document ထဲမှာ top most element ဖြစ်တဲ့အတွက် သူဟာ :root element ပဲဖြစ်ပါတယ်။ ကျွန်တော်တို့ :root ဆိုတဲ့ CSS rule ထဲမှာ ကျွန်တော်တို့ပေးချင်တဲ့ name နဲ့ color variable နှစ်ခုကိုရေးလိုက်မယ်။ Eg, :root { --color-primary: #278da4; --color-secondary: #b13c69; } ။ ဒါကြောင့် root rule အတွင်းမှာ custom property names တွေရေးခြင်းဟာ properties တွေကို CSS cascade မှာ highest level ဖြစ်စေပါတယ်။ အခု values တွေကို document ထဲမှာ element အားလုံးကနေ inherited လုပ်ထားနိုင်ပြီး သူတို့ကို style sheet ထဲက ဘယ်နေရာကနေမဆို reference လုပ်နိုင်ပါတယ်။ ဒါဟာ SaaS or JavaScript မှာလို global variables တွေကို declare လုပ်တာနဲ့ဆင်တူပါတယ်။ ဘာလို့လဲဆိုရင် values တွေကို နေရာတိုင်းမှာရနိုင်လို့ဖြစ်ပါတယ်။ အခု browser ဟာ ပထမဆုံး custom property တစ်ခုနဲ့တော့ ဘာမှအလုပ်မလုပ်ပါဘူး။ Custom property တစ်ခုရဲ့ value ကအမှန်တစ်ကယ် အလုပ်လုပ်ဖို့ဆိုရင် var( ) function ကိုအသုံးပြုပေးရပါမယ်။ ဘယ် property ရဲ့ value မှာမဆို var( ) function တစ်ခုမက ပါဝင်နိုင်ပါတယ်။ CSS variables တွေကို font stacks, layout styles နဲ့ duplication တွေကို avoid လုပ်ဖို့လည်းအသုံးပြုနိုင်ပါတယ်။

Example of variables.css File:

        
            /* VARIABLES ------------------------------ */

            :root {
                --color-primary: #278da4;
                --color-secondary: #b13c69;

                --color-bg: #3acec2;
                --color-bg-light: #009fe1;

                --gradient: var(--color-bg-light),
                            var(--color-bg);
                
                --font-stack-primary: 'Raleway', sans-sefif;
                --font-stack-secondary: 'Bree Serif', serif;

                --max-width: 1000px;
                --gutter: 12px;
            }

            /* BASE STYLES ---------------------------- */

            body {
                font-family: var(--font-stack-primary;
            }

            h1,
            h2 {
                font-family: var(--font-stack-secondary);
            }

            a,
            h2 {
                color: var(--color-primary);
            }

            header {
                background: linear-gradient(var(--gradient)), url('../img/bg.jpg') no-repeat;
                background-blend-mode: multiply;
                background-size: cover;
            }

            /* CONPONENT STYLES ------------------------------ */

            .main-nav li {
                margin: 0 var(--gutter);
            }

            .btn-callout {
                background-color: var(--color-secondary;
            }

            .btn-info {
                background-color: var(--color-primary);
            }

            /* MEDIA QUERIES -------------------------------- */

            @media (min-width: 768px) {
                .main-content {
                    max-width: var(--max-width);
                }
            }

CSS variables တွေဟာ preprocessor variables တွေနဲ့မလုပ်နိုင်တဲ့ အရာတော်တော်များများကို အလုပ်လုပ်ပေးနိုင်ပါတယ်။ သူတို့တစ်ခုချင်းစီမှာ အမျိုးမျိုးသော circumstances တွေအောက်မှာ ရှိပါတယ်။ Preprocessors တွေက browser ထဲမှာ run မပေးနိုင်ဘူးဆိုတာ တွေ့ရမှာပါ။ ကျွန်တော်တို့ declare လုပ်တဲ့ variables တွေဟာ ကျွန်တော်တို့စက်ထဲက source file တစ်ခုအတွင်းမှာရှိပါတယ်။ Preprocessors တွေက သူတို့ CSS နဲ့ compile လုပ်တဲ့အချိန် variables တွေကို static values တွေနဲ့ replace လုပ်ပါတယ်။ Browser က variables တွေအကြောင်းကို လုံးဝမသိဘဲ အဲ့ဒီ variables တွေကိုယ်တိုင်ကလည်း ကျွန်တော်တို့ markup ကို သိမှာမဟုတ်ပါဘူး။ CSS variables တွေဟာ compiled လုပ်ထားတဲ့ code တွေထက် ပိုပြီးတော့ flexible ဖြစ်ပါတယ်။ ဘာလို့လဲဆိုရင် သူတို့က browser အတွင်းမှာရှိနေလို့ဖြစ်ပါတယ်။ CSS variable ရဲ့ value ကို update လုပ်တဲ့အခါ browser ကသူ့ရဲ့ runtime မှာ အဲ့ဒီ value ကို ဘယ်နေရာမှာပဲ reference လုပ်နေပါစေ update လုပ်ပေးမှာဖြစ်ပါတယ်။ ဒါကြောင့် CSS variable တွေကို dynamically အရ update or alter လုပ်လို့ရတာဖြစ်ပါတယ်။

ကျွန်တော်တို့အနေနဲ့ CSS variable တစ်ခုသတ်မှတ်ဖို့အတွက် သူ့ရဲ့ scope ကို define လုပ်ရတာကို လေ့လာခဲ့ပါတယ်။ ပြီးတော့ variable တစ်ခုကို :root အတွင်းမှာ scope လုပ်ခြင်းဟာ အဲ့ဒီ variable က elements အားလုံးမှာ available ဖြစ်တယ်လို့ပြောချင်တာဖြစ်ပါတယ်။ ဒါကြောင့် root ကိုသုံးခြင်းဟာ global variables တွေ set လုပ်ဖို့ကောင်းတဲ့ နည်းလမ်းဖြစ်ပြီး ကျွန်တော်တို့ရဲ့ variables အားလုံးကို တစ်နေရာတည်းကနေ maintain လုပ်နိုင်မှာဖြစ်ပါတယ်။ :root ရဲ့အတွင်းမှာ variables တွေကို အမြဲတမ်းကြီးလည်း define လုပ်ဖို့မလိုအပ်ပါဘူး။

Var function ဟာ fallback value တစ်ခုအဖြစ်လုပ်ဆောင်မယ့် optional second argument တစ်ခုကိုလည်း accept လုပ်ပေးပါတယ်။ ကျွန်တော်တို့ CSS ထဲမှာ CSS variables တွေကို ဘယ်နေရာမှ မကြေငြာရသေးဘူး (သို့) referenced variable က invalid ဖြစ်နေမယ်ဆိုရင် second argument က var ကို variable အတွက် substitution value တစ်ခုအဖြစ် အစားထိုးပြီး pass လုပ်ပေးသွားမှာဖြစ်ပါတယ်။ ဒါကြောင့် second argument (သို့) fallback တစ်ခုကို comma ခြားပြီး 0.85em လို့ရေးလိုက်ပါမယ်။ Fallback values တွေပေးထားခြင်းဟာ ကျွန်တော်တို့ကို elements တွေအတွက် defensive styles တွေရေးဖို့ ခွင့်ပြုပေးပြီး ၊ ကျွန်တော်တို့ရဲ့ style sheet ကိုလည်း sharable နဲ့ extensible လုပ်နိုင်ပါတယ်။ For example- ကျွန်တော်တို့က initial design ကို var function နဲ့ fallback value ကိုတွဲသုံးပီးတော့ create လုပ်နိုင်ပါတယ်။ ကျွန်တော်တို့က CSS variable ကိုသုံးပြီးတော့ calculation ကိုအသုံးပြုလို့ရပါတယ်။ Length value ကို determine လုပ်ဖို CSS calc function ကိုအသုံးပြုနိုင်ပါတယ်။ calc function နဲ့ var function combine လုပ်တာကို အောက်က example မှာကြည့်နိုင်ပါတယ်။

                ball.style.setProperty('--pos-x', e.clientX);
                ball.style.setProperty('--pos-y', e.clientY);
                ball.style.setProperty('--ball-bg', randomHex());

            // set a random hex value
            function randomHex() {
                let max = 1 << 24;
                let hex = (max + Math.floor(Math.random() * max)).toString(16).slice(-6);
                return `#${hex}`;
            }
