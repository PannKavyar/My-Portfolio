CSS Transition Properties

CSS transitions ဟာ ကျွန်တော်တို့ရဲ့ interface ရဲ့ အစိတ်အပိုင်းတွေကို အချိန်တစ်ခု (period of time) အတွင်းမှာ value တစ်ခုကနေတစ်ခုကို set of properties တွေကို change လုပ်ခြင်းဖြင့် animate လုပ်ပေးနိုင်ပါတယ်။ ပုံမှန်အရ ကျွန်တော်တို့ button တစ်ခုကို hover လုပ်တဲ့အခါ အရောင်ပြောင်းဖို့အတွက် CSS property တစ်ခုရဲ့ value ကိုပြောင်းပေးရပါတယ်။ ဒါပေမယ့် CSS transition နဲ့ဆိုရင် အဲ့ဒီ button ရဲ့ animation ကို ဖြည်းဖြည်းနဲ့ ငြင်သာစွာ ပြောင်းလဲသွားအောင် လုပ်နိုင်ပါတယ်။ Value တစ်ခုကနေ တစ်ခုကို CSS properties တွေနဲ့ transition လုပ်ဖို့အတွက်ဆိုရင် ကျွန်တော်တို့က browser ကို ဘယ် propeties တွေက transition လုပ်ဖို့နဲ့ အဲ့ဒီ transition effect က ဘယ်လောက်ကြာမလဲဆိုတာ ပြောပေးရပါမယ်။ ဒါဆိုရင်တော့ CSS အနည်းငယ်နဲ့ browser က delay ဖြစ်မယ့် transition ကို control လုပ်နိုင်ပါပြီ။ Transitions တွေဟာ ပုံမှန်အားဖြင့် user action တစ်ခုပါပဲ။ ဆိုလိုတာက user က mouse click or hover, focus နဲ့ active တွေကို နှိပ်ပေးမှသာ ပြောင်းလဲသွားတာမျိုးဖြစ်ပါတယ်။ ဒါကြောင့် ပထမဆုံးအနေနဲ့ ကျွန်တော်တို့က transition duration property နဲ့ transition တစ်ခုရဲ့ length ကိုပဲ သတ်မှတ်ပြီးတော့ transition တစ်ခုကို ရိုးရိုးရှင်းရှင်းလေး ပြုလုပ်သွားမှာဖြစ်ပါတယ်။

Perspective-origin Property

အစောပိုင်းက ကျွန်တော်တို့ content rule ထဲမှာ content div ကို 3D space တစ်ခုရဖို့အတွက် perspective value ကို 700px ပေးခဲ့ပါတယ်။ Default အနေနဲ့ viewer မြင်ရမယ့် 3D space ရဲ့ position ကတော့ screen ရဲ့ center ပဲဖြစ်ပါတယ်။ အခုဆိုရင်တော့ rotations တွေကို ကျွန်တော်တို့မြင်နေရပါတယ်။ ဒါပေမယ့် အဲ့ဒီ viewing position ကို perspective-origin property နဲ့ change လုပ်နိုင်ပါတယ်။ တစ်ကယ်လို့ ကျွန်တော်တို့ image ရဲ့ scene ကို real life မှာမြင်ချင်တယ်ဆိုရင် ဒီလိုမျိုး direct centered ကျနေတဲ့ angle ကနေတော့ မြင်ရမှာမဟုတ်ပါဘူး။ perspective-origin က value နှစ်ခုကို accept လုပ်ပါတယ်။ Horizontal နဲ့ vertical position ဆိုပီးတော့ accept ဖြစ်တာပါ။ သူဟာ length units, percentages or position keyword, top, right, bottom နဲ့ left တွေကိုလည်း accept လုပ်ပေးနိုင်ပါတယ်။ Viewing position ရဲ့ default က screen ရဲ့center မှာဖြစ်သောကြောင့် perspective-origin က 50%,50% ဖြစ်ပါတယ်။ ကျန်တဲ့ value တွေကိုတော့ example မှာနိုင်ပါတယ်။
