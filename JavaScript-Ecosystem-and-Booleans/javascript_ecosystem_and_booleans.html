ကျွန်တော်တို့ဟာ applications နဲ့ tools တွေကို built လုပ်လို့ရပါတယ်။ နောက်ထပ် popular ဖြစ်နေတဲ့ JavaScript frameworks တွေနဲ့ libraries တွေများစွာရှိနေပါသေးတယ်။ JavaScript ကို အစောပိုင်းတည်းက browser
