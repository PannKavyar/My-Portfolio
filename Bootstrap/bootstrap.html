Introduction to Bootstrap

ကျွန်တော်တို့ဟာ website or app တစ်ခု စပြီး တည်ဆောက်ပြီဆိုရင် ဘယ် front-end tools တွေကို သုံးသင့်လည်းဆိုတာကို မကြာခဏစဉ်းစားရပါတယ်။ Layout ကိုဘယ်လိုချမလဲ၊ floats, grid system or flexbox မှာရှိတဲ့ venture စတာတွေကို သုံးသင့်ရဲ့လား။ Typography နဲ့ media queries တွေအတွက် pixels (သို့) ems တွေကို သုံးရမလား။ Browser compatibility ကရော ဘယ်လိုလဲ?။ စသဖြင့် အမျိုးမျိုး စဉ်းစားရပါတယ်။ Website တစ်ခု (သို့) app တစ်ခုရဲ့ အဓိကအကျဆုံးအပိုင်းကလည်း အဲ့ဒီ website ကို ဘယ်လိုစပြီး design ချရမလဲဆိုတာပါပဲ။ ဒါကြောင့် ဒီသင်ခန်းစာမှာတော့ ကျွန်တော်တို့က functional design နဲ့ layout ကိုတည်ဆောက်ရာမှာ အထောက်အကူဖြစ်ပြီး popular အဖြစ်ဆုံး open source front-end frameworks တွေထဲက တစ်ခုဖြစ်တဲ့ Bootstrap အကြောင်းကို လေ့လာသွားမှာဖြစ်ပါတယ်။ Bootstrap ကိုအသုံးပြုရတဲ့အထဲက အကြီးမားဆုံး advantage ကတော့ development speed ပဲဖြစ်ပါတယ်။ Bootstrap ဟာ latest အဖြစ်ဆုံးနဲ့ အသုံးအများဆုံး advanced front-end tools တွေကိုသုံးထားတဲ့အတွက် ကျွန်တော်တို့ရဲ့ web development ကို faster & easier ဖြစ်စေပါတယ်။ ကျွန်တော်တို့ app ရဲ့ design နဲ့ coding ပိုင်းကို custom scratch ကနေယူပြီး ရေးနေစရာမလိုဘဲ Bootstrap ရဲ့ ready-made components တွေကို သုံးပြီး လွယ်ကူစွာ ရေးနိုင်မှာဖြစ်ပါတယ်။ Bootstrap framework ဟာ buttons, forms, navigation, grids ကဲ့သို့သော common elements တွေအတွက် HTML နဲ့ CSS ကို reusable အနေနဲ့လာပါတယ်။ ပြီးတော့ ကျွန်တော်တို့အနေနဲ့ JavaScript code တွေကို တစ်ကြောင်းချင်းစီရေးနေစရာမလိုဘဲ Bootstrap မှာပါတဲ့ custom JavaScript plugins တွေကိုသုံးပြီး components တွေအများကြီးနဲ့ interact လုပ်နိုင်မှာဖြစ်ပါတယ်။ Bootstrap ရဲ့ အကောင်းဆုံးအပိုင်းကတော့ cross-browser compatibility ပဲဖြစ်ပါတယ်။ ကျွန်တော်တို့ website က IE, Safari, Firefox နဲ့ Chrome တွေမှာ responsive ဖြစ်ဖို့အတွက် အချိန်အများကြီးပေးနေစရာမလိုဘဲ Bootstrap က အဲ့ဒီ responsive အပိုင်းတွေကို တစ်ခါတည်း ထည့်ပေးထားပါတယ်။ ဒီအတွက်ကြောင့် ကျွန်တော်တို့ရေးလိုက်တဲ့ code တွေက phones, tablets နဲ့ desktop devices တွေအားလုံးမှာ အလုပ်လုပ်နိုင်မှာဖြစ်ပါတယ်။ တစ်ခုရှိတာကတော့ bootstrap ကိုလေ့လာမယ်ဆိုရင် HTML နဲ့ CSS ရဲ့ အခြေခံသဘောတရားအားလုံးကို နားလည်ထားဖို့လိုပါတယ်။

Using Bootstrap

Bootstrap ကို Twitter မှာရှိတဲ့ developers တွေက 2011 မှာ create လုပ်ခဲ့ကြတာဖြစ်ပါတယ်။ Bootstrap ဟာ web ပေါ်က building, responsive နဲ့ mobile first projects တွေအတွက် popular အဖြစ်ဆုံး HTML, CSS နဲ့ JavaScript frameworks တွေထဲက တစ်ခုဖြစ်ပါတယ်။ Web ပေါ်မှာ cross ဖြစ်နေတဲ့ 7 million လောက်ရှိတဲ့ apps တွေနဲ့ sites တွေဟာ Bootstrap ကိုသုံးပြီး built လုပ်ထားတာဖြစ်ပါတယ်။ Web designers, front-end developers နဲ့ programmers အများစုကလည်း projects တွေတည်ဆောက်တဲ့အခါမှာ မြန်ဆန်လွယ်ကူစေဖို့အတွက် Bootstrap ကိုပဲ အသုံးပြုကြတာဖြစ်ပါတယ်။ NASA, FIFA, Spotify နဲ့ Lyft လိုမျိုး popular companies နဲ့ agencies တွေတောင်မှာ သူတို့ရဲ့ websites တွေကို Bootstrap နဲ့ အသုံးပြုထားတာဖြစ်ပါတယ်။ Bootstrap ဟာ front-end framework တစ်ခုဖြစ်ပြီး ဘာကိုဆိုလိုသလဲဆိုရင် Framework တစ်ခုဟာ ကျွန်တော်တို့ကို higher level projects တွေကို lower level နဲ့ repetitive details တွေကို စိုးရိမ်စရာမလိုဘဲ တည်ဆောက်တဲ့အခါမှာ ကူညီပေးတဲ့ reusable platform တစ်ခုဖြစ်ပါတယ်။ ဥပမာပြောရရင် ကျွန်တော်တို့ website or app တစ်ခုတည်ဆောက်တဲ့အခါ ကျွန်တော်တို့ website ကို devices အများစုနဲ့ screen sizes အမျိုးမျိုးမှာ ဖော်ပြလိုက်တဲ့အခါ အဲ့ဒီ layout ထဲက nav bars, buttons, headers, forms, modals လိုမျိုး UI components တွေဟာ lost ဖြစ်သွားနိုင်ပါတယ်။ ဒါဆိုရင် ကျွန်တော်တို့က screen sizes တွေတိုင်းအတွက် custom solution တွေနဲ့ website ကို create လုပ်နေရပါလိမ့်မယ်။ အဲ့ဒီလိုလုပ်မယ့်အစား Bootstrap ဆိုတဲ့ framework ကိုအသုံးပြုခြင်းဖြစ်ပါတယ်။ Front-end framework တစ်ခုဟာ typography နဲ့ browser compatibility problems တွေကို အစတည်းက ဖြေရှင်းနေရပါတယ်။ ဒါကြောင့် Bootstrap ကိုသုံးခြင်းအားဖြင့် ကျွန်တော်တို့ရဲ့ content (သို့) design ကို ပိုပြီး focus လုပ်နိုင်မှာဖြစ်ပါတယ်။ Bootstrap ရဲ့ latest documentation ကို getbootstrap.com မှာ လေ့လာနိုင်ပါတယ်။ အစောကပြောခဲ့သလိုပဲ Bootstrap ဟာ website အများစုနဲ့ app projects တွေမှာသုံးတဲ့ user interface tasks တွေအတွက် common content layout styles တွေရဲ့ code တွေသာမက JavaScript code components တွေကိုပါ provide လုပ်ပေးထားပါတယ်။ Developer အချို့ဟာ Bootstrap ကို predefined design styles နဲ့ grid တွေအတွက် အသုံးပြုပါတယ်။ ဒါကြောင့် သူတို့အနေနဲ့ rock solid responsive layout တစ်ခုနဲ့ well-presented site တစ်ခုကို မြန်ဆန်စွာ build လုပ်နိုင်မှာဖြစ်ပါတယ်။ အချို့တွေကတော့ သူ့တို့ရဲ့ page ကို grid တွေနဲ့ layout ချရုံလောက်သုံးပြီး CSS ကိုတော့ custom CSS နဲ့ design လုပ်တာဖြစ်ပြီး အချို့ကတော့ JavaScript components တွေအတွက်ပဲ သုံးကြပါတယ်။ ဒီသင်ခန်းစာမှာတော့ Bootstrap ရဲ့ popular ဖြစ်တဲ့ Nav bar, Jumbotron buttons, forms တွေနဲ့ full fledged website design အချို့ကို built လုပ်သွားမှာဖြစ်ပါတယ်။ Bootstrap ကိုသုံးပြီးရေးထားတဲ့ websites တွေကိုလေ့လာချင်တယ်ဆိုရင်တော့ expo.getbootstrap.com မှာ လေ့လာနိုင်ပါတယ်။

Creating SignUp Form

Bootstrap ကို အလွယ်ကူဆုံးသုံးလို့ရမယ့် နည်းလမ်းအချို့ရှိပါတယ်။ Bootstrap file ကို download လုပ်မယ်ဆိုရင်တော့ getbootstrap.com မှာ download ရယူနိုင်ပါတယ်။ Download မလုပ်ချင်ရင်တော့ အမြန်ဆုံးနည်းဖြစ်တဲ့ Bootstrap CDN(Content Delivery Network) ဆိုတဲ့ link ကို ရယူနိုင်ပါတယ်။ ဒီလို CDN links ကို HTML file ထဲမှာ ထည့်ခြင်းဖြင့် CDN servers ကနေ CSS နဲ့ JavaScript တွေကို တစ်ခါတည်း load လုပ်နိုင်မှာဖြစ်ပါတယ်။ ဒါကြောင့် အခုကျွန်တော်တို့ သင်ခန်းစာမှာ Bootstrap CDN link ကိုသုံးပြီး တည်ဆောက်သွားမှာဖြစ်ပါတယ်။ Bootstrap ဟာ global styles တွေအများကြီးနဲ့လာပြီးတော့ ကျွန်တော်တို့ content တွေကိုလည်း settings တွေလုပ်နိုင်ပါသေးတယ်။ ဥပမာပြောရရင် သူဟာ browsers အမျိုးမျိုးနဲ့ operating systems တွေကို တူညီစွာ across လုပ်ပြီး HTML tags တွေကို render လုပ်ပေးတဲ့ base styles တွေအများကြီးပါတဲ့ Normalized CSS reset ကိုအသုံးပြုပါတယ်။ ပြီးတော့ သူက Reboot လို့ခေါ်တဲ့ custom set of styles တစ်ခုနဲ့ base CSS rules တွေပေါ်မှာ build လုပ်လို့ရပါသေးတယ်။ Normalize နဲ့ Reboot ကတော့ ကျွန်တော်တို့ project ကို Chrome, Firefox, Safari နဲ့ IE လိုမျိုး browsers တွေမှာ အတူတူဖြစ်အောင် run ပေးပါတယ်။ Bootstrap နဲ့ကျွန်တော်တို့က heading, body text, lists နဲ့ အခြားအရာတွေကို customize လုပ်နိုင်ပါတယ်။ Container တွေကတော့ Bootstrap ထဲမှာ basic layout element ဖြစ်ပါတယ်။ Bootstrap container ကတော့ fixed or fluid ဖြစ်နိုင်ပါတယ်။ ဒါကြောင့် class container-fluid ကတော့ full width container တစ်ခုကို create လုပ်ပေးပါတယ်။ Bootstrap ကတော့ element တွေကို custom styles တွေနဲ့ provide လုပ်ပေးနိုင်ပါတယ်။

            <!doctype html>
            <html lang="en">

            <head>
                <title>Full Stack Conf</title>
                <!-- Required meta tags -->
                <meta charset="utf-8">
                <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">

                <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.5.0/css/bootstrap.min.css">
            </head>

                            </span>

Styling Form with Bootstrap

Bootstrap ရဲ့ class တစ်ခုနဲ့ ကျွန်တော်တို့ website ထဲက background color ကို change လုပ်နိုင်ပါတယ်။ ဒီလိုလုပ်ဖို့အတွက် Utilities ထဲက colors ကိုသွားပြီး bg-primary ကို copy လုပ်ပြီးတော့ ကျွန်တော်တို့ body ထဲမှာထည့်လိုက်ရုံနဲ့ ကျွန်တော်တို့ website ရဲ့ background color က primary color အဖြစ်ပြောင်းလဲသွားမှာဖြစ်ပါတယ်။ ကျွန်တော်တို့ body ထဲမှာရှိတဲ့ text တွေကို အရောင်ပြောင်းချင်တယ်ဆိုရင်တော့ text-white ဆိုတဲ့ class နဲ့ ပြောင်းပေးရပါမယ်။ နောက်ပြီးတော့ getbootstrap.com ထဲက Utilities ထဲက spacing ထဲမှာ ကျွန်တော်တို့ page ရဲ့ margin နဲ့ padding တွေအတွက် shorthand responsive margin & padding တွေရှိပါတယ်။ အဲ့ဒီမှာရှိတဲ့ class name တွေနဲ့ ကျွန်တော်တို့ရဲ့ content တွေကို vertical, horizontal, padding နဲ့ margin တွေသတ်မှတ်ပေးနိုင်ပါတယ်။ Spacing မှာ t, b, l, f, x, y ဆိုပြီး top, bottom, left, right, both left & right နဲ့ both top & bottom ဆိုပြီး shorthand တွေကို ဖော်ပြပေးထားပါတယ်။ Bootstrap ဟာ margins နဲ့ padding တွေအတွက် CSS rem unit ကိုလည်း အသုံးပြုပါတယ်။ Rem ဆိုတာကတော့ root အတွက် page ရဲ့ root element<html> ရဲ့ font size နဲ့ အမြဲတမ်း relative ဖြစ်နေတဲ့ flexible CSS unit တစ်ခုဖြစ်ပါတယ်။ Bootstrap ရဲ့ global default value ကတော့ 1rem ပဲဖြစ်ပါတယ်။ ဥပမာ ကျွန်တော်တို့က margin or padding ကို 1rem ကနေ 0.25rem ၊ 2rem ဆိုရင် 0.5rem နဲ့ 4rem ကနေ 1.5rem ၊ 5rem ဆိုရင် 3rem ဖြစ်သွားမှာဖြစ်ပါတယ်။ ml-1 ဆိုတာကတော့ margin-left 1 ဖြစ်ပါတယ်။ ဆိုလိုတာကတော့ margin-left 1 ကိုပြောတာဖြစ်ပါတယ်။ ဒါကြောင့် margin value က 0.25rem ရှိနေမှာဖြစ်ပါတယ်။

Responsive and Media Queries in Bootstrap

Mobile Responsive

Mobile first grid ကတော့ 12 column layout ပေါ်ကို အခြေခံထားတာဖြစ်ပါတယ်။ Bootstrap မှာ grid tiers ငါးခုရှိပါတယ်။ .col-xs ကတော့ extra small display အတွက်ဖြစ်ပါတယ်။ .col-sm-ကတော့ landscape view နဲ့ phone အတွက်ဖြစ်ပါတယ်။ .col-md- ကတော့ medium display အတွက် tablet လိုမျိုးဖြစ်ပြီးတော့ .col-lg- ကတော့ desktop displayအတွက်ဖြစ်ပါတယ်။ .col-xl- ကတော့ extra large desktop display အတွက်ဖြစ်ပါတယ်။ ကျွန်တော်တို့က column တွေအတွက် order class တွေသုံးနိုင်ပါတယ်။ order-lg-1 ဆိုတာကတော့ column ကို row ရဲ့ပထမဆုံးကိုထားပေးမှာဖြစ်ပါတယ်။ ဘာလို့လဲဆိုတော့ ကျွန်တော်တို့က flexbox order က အလုပ်လုပ်ပေးလို့ဖြစ်ပါတယ်။ order-lg-2 ဆိုရင် second position မှာဖြစ်ပြီးတော့ order-lg-3 ဆိုရင်တော့ ကျွန်တော်တို့က third position မှာဖြစ်ပါတယ်။ ကျွန်တော်တို့က image ကို extra display အတွက် image ကို hide ပေးထားချင်ရင်တော့ d-none ပေးရမှာဖြစ်ပါတယ်။ တကယ်လို့ extra large အတွက် image ကိုပြန်ပေါ်ချင်ရင်တော့ ကျွန်တော်တို့က d-sm-block ပေးခဲ့ရမှာဖြစ်ပါတယ်။ col-xl-5 ကတော့ extra lage လို့ဆိုလိုပြီး column တွေခွဲပေးတာဖြစ်ပါတယ်။ ml auto ဆိုတာကတော့ပိုနေတဲ့ left ဘက်က margin ကိုနေရာယူပေးသွားတာဖြစ်ပါတယ်။ Bootstrap ကတော့ complex or time consuming task တွေလိုမျိုး navigation systems, form controls, drop down menu နဲ့တခြားအရာတွေလိုမျိုးကို simplifies ဖြစ်စေပါတယ်။ Interactive components တွေဖြစ်တဲ့ drop down, alert နဲ့ modal window တွေကတော့ JavaScript နဲ့အလုပ်လုပ်ပါတယ်။ Bootstrap ဟာ interactive component တွေကို class, id or data attribute နဲ့ create လုပ်နိုင်ပါတယ်။ Bootstrap တော့ JavaScript ရေးဖိုမလိုပဲ provide လုပ်ပေးပါတယ်။ ကျွန်တော်တို့ အခု navbar တစ်ခုကို create လုပ်ကြည့်ပါမယ်။ nav bar မှာ ဝင်ကြည့်လို့ရပါတယ်။ နောက်တစ်ခုကတော့ navbar ဆိုတာ navigation, branding နဲ့တခြား elements တွေကို navigation header တစ်ခုမှာ positioning လုပ်တဲ့ wrapper တစ်ခုဖြစ်ပါတယ်။
