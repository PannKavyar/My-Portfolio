ကိုသုံးပြီးတော့ content တွေရဲ့ အဓိကကျတဲ့ sections တွေကို layout ချဖို့နဲ့ responsive forms နဲ့ media objectsတွေလိုမျိုး သေးငယ်တဲ့ screen တွေမှာ အဆင်ပြေစွာမြင်ရဖို့အတွက် သေးငယ်တဲ့ user-interface elements တွေတည်ဆောက်တဲ့အခါမှာ ဒီ CSS grid တွေအသုံးပြုပြီးတော့ တည်ဆောက်ကြပါတယ်။ တစ်ချို့ site တွေမှာဆိုရင် site တစ်ခုလုံးကို grid ကိုပဲသုံးပြီး design ချထားတာတွေတောင်ရှိပါတယ်။ လာမယ့်သင်ခန်းစာမှာတော့ Grid Concepts

ကျွန်တော်တို့ဟာ CSS Grid နဲ့ layout မတည်ဆောက်ခင်မှာ CSS Grid ရဲ့concepts အချို့နဲ့ သူ့နောက်မှာအလုပ်လုပ်ပေးတဲ့ terminology တို့ကို နားလည်ထားသင့်ပါတယ်။ ဒီသင်ခန်းစာမှာ အဲ့ဒီနှစ်ခုကို အဓိကထားပြီးပြောပြသွားပါမယ်။ CSS Grid Layout မှာရှိတဲ့ အရေးအကြီးဆုံး elements နှစ်ခုကတော့ grid containers နဲ့ grid items ပဲဖြစ်တယ်။ Grid ကိုသုံးမယ်ဆိုရင် ပထမဆုံး သတ်မှတ်ပေးရမှာကတော့ grid container ဖြစ်ပါတယ်။ Grid container ဟာ grid formatting context ကို တည်ဆောက်ပေးပြီး သူ့ရဲ့ direct child elements တွေ ဘယ်လို spaced, sized နဲ့ aligned လုပ်မယ်ဆိုတာကို manage လုပ်ပေးပါတယ်။ ကျွန်တော်တို့ element တစ်ခုကို grid container တစ်ခု create လိုက်မယ်ဆိုရင် အဲ့ဒီ element ရဲ့ direct children တွေအားလုံးဟာ grid items တွေဖြစ်သွားပါတယ်။ ပြီးတော့ grid container ထဲမှာ grid items အရေအတွက်ဟာ အများကြီးရှိနိုင်ပါတယ်။ အောက်မှာပြထားတဲ့ ဥပမာမှာဆိုရင် main element ဟာ grid container ဖြစ်မယ်။ ပြီးတော့ သူရဲ့ တိုက်ရိုက် children ဖြစ်တဲ့ article elements တွေက grid items တွေဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ main element အတွင်းမှာရှိတဲ့ headings, paragraphs နဲ့ images တွေလိုမျိုး အခြား elements တွေကို nested လုပ်ချင်တယ်ဆိုရင်လည်း သူတို့ဟာ grid items တွေဖြစ်သွားမှာဖြစ်ပါတယ်။ ဒါကြောင့် grid formatting context ကိုဆောက်ပြီးသွားပြီဆိုရင် သူ့ရဲ့ content ကို grid အဖြစ် positioning နဲ့ aligning လုပ်နိုင်ပါတယ်။ အဲ့ဒီနောက်မှာတော့ အဲ့ဒီ grid ဟာ grid lines လို့ခေါ်တဲ့ horizontal နဲ့ vertical lines တွေလိုပုံစံဖြစ်သွားမှာဖြစ်ပါတယ်။ Grid lines တွေဟာ ကျွန်တော်တို့ grid container ရဲ့ space တွေကို rows နဲ့ columns တွေအတိုင်း (သို့မဟုတ်) grid items တွေကို aligned, sized နဲ့ placed တွေလုပ်ဖို့အတွက် areas တွေကို divide လုပ်ပေးလိုက်ပါတယ်။ Horizontal grid lines တွေဟာ rows တွေအဖြစ် နေရယူသွားပြီး ၊ vertical grid lines တွေဟာ columns တွေအဖြစ်နဲ့ နေရာယူသွားမှာဖြစ်ပါတယ်။ Grid line ဟာ column or row တစ်ခုရဲ့ side တစ်ဖက်ဖက်မှာ တည်ရှိပါတယ်။ တစ်နည်းအားဖြင့်ဆိုရမယ်ဆိုရင် ကပ်လျက်ရှိနေတဲ့ grid lines နှစ်ခုကြားက space ဟာ column or row အနေနဲ့ ပုံစံချပေးတာဖြစ်ပါတယ်။

Grid Container

CSS grid ရဲ့ concepts နဲ့ သူ့နောက်မှာအလုပ်လုပ်ပေးတဲ့ terminology ကိုသိပြီးတဲ့နောက်မှာ ကျွန်တော်တို့ဟာ grid တစ်ခုဖြစ်တဲ့ grid containerကို columns နဲ့ rows တွေသတ်မှတ်ပြီး တည်ဆောက်လို့ရပါပြီ။ အခုသင်ခန်းစာမှာတော့ container ကိုတည်ဆောက်ဖို့ grid display value တွေဖြစ်တဲ့ grid ရဲ့ related properties တွေဖြစ်တဲ့ columns ကိုသတ်မှတ်တဲ့ grid-template-columns နဲ့ rows ကိုသတ်မှတ်တဲ့ grid-template-rows တွေအကြောင်းကို ပြောပြပေးသွားမှာဖြစ်ပါတယ်။ အောက်မှာပြထားတဲ့ example အတိုင်း class grid တစ်ခုဖြစ်တဲ့ div ထဲမှာ nested ဖြစ်နေတဲ့ div သုံးခုကို class name အနေနဲ့ item ဆိုပြီး ထည့်မှာဖြစ်ပါတယ်။ ကျွန်တော်တို့ CSS folder ထဲမှာ page.css နဲ့ grid.css နှစ်ခုတည်ဆောက်ထားမယ်။ page.css ထဲမှာ page ရဲ့ basic style တွေပါဝင်ပြီးတော့ grid.css ထဲမှာတော့ grid layout ကိုရေးမှာဖြစ်ပါတယ်။ CSS grid layout တစ်ခုမှာ grid container တစ်ခုနဲ့စပါတယ်။ Grid container ကိုတော့ element ရဲ့ grid property ကို display: grid; လို့ setting ချရမှာဖြစ်ပါတယ်။ ဒါကြောင့် div ထဲက class name ပေးထားတဲ့ grid ဟာ grid container Grid items တွေကို layout ချဖို့အတွက် grid tracks တွေကို ကျွန်တော်တို့ create လုပ်နိုင်သော်လည်း CSS Grid ဟာ အမှန်တစ်ကယ် grid items တွေရဲ့ placement တွေကို သတ်မှတ်ခြင်းဖြင့် control လုပ်ဖို့ ခွင့်ပြုပေးပါတယ်။ ကျွန်တော်တို့ဟာ grid item တစ်ခုကို properties တွေတိုက်ရိုက်ထည့်ခြင်းဖြင့် items တွေဘယ်လို placed လုပ်ရမလဲဆိုတာ control လုပ်ဖို့ grid line numbers တွေကိုသုံးနိုင်ပါတယ်။ ဒါဟာ grid item က ဘယ်ကစပြီး ဘယ်မှာဆုံးသွားတယ်ဆိုတာနဲ့ grid tracks တွေဘယ်လောက်များများ expand လုပ်သလဲဆိုတာကို specify လုပ်နိုင်တဲ့အတွက် grid item ရဲ့ placement, size နဲ့ order တွေကို control လုပ်နိုင်ပါတယ်။ ဥပမာအနေနဲ့ အခု line based placement ကိုသုံးပြီး page layout ရဲ့ header, main, aside နဲ့ footer တို့ကို create လုပ်ပြသွားမှာဖြစ်ပါတယ်။ ကျွန်တော်တို့ index.html ထဲမှာ container ဆိုတဲ့ class တစ်ခုပါတဲ့ div တစ်ခုရှိမယ်။ သူ့ထဲမှာ nested လုပ်ထားတဲ့ header, main, aside နဲ့ footer elements တွေပါမယ်။ grid-column-start ဆိုတဲ့ property ဟာ element စမယ့် grid column line ကိုပြပေးတာဖြစ်ပြီး grid-column-end ဆိုတဲ့ property ကတော့ element ဆုံးမယ့် grid column line ကိုပြတာဖြစ်ပါတယ်။ vh ဆိုတာ လက်ရှိ viewport height ရဲ့ 100% ကိုဆိုလို့တာဖြစ်ပါတယ်။ Summary အနေနဲ့ line-based placement properties တွေဖြစ်တဲ့ grid-column-start နဲ့ grid-row-start ဟာ item စမယ့် line ကို specify လုပ်ပေးတာဖြစ်ပြီး ၊ grid-column-end နဲ့ grid-row-end ကတော့ item ရဲ့ ဆုံးမယ့် line ကို specify လုပ်တာဖြစ်ပါတယ်။
