ကျွန်တော်တို့ ဒီတစ်ခါပြောပြသွားမှာကတော့ Flexible layout or Flexbox feature ပဲဖြစ်ပါတယ်။ ဒါကြောင့် ကျွန်တော်တို့ပြောခဲ့သလို CSS ဟာ layout တွေကို handle လုပ်ဖို့အတွက် great language ဆိုပြီး တိတိကျကျမရှိပါဘူး။ ဥပမာပြောရမယ်ဆိုရင် floats တွေမှာ common layout method တစ်ခုရှိပေမယ့်လည်း အဲ့ဒါတွေက complicated ဖြစ်တဲ့ box model တွေပဲဖြစ်ပါတယ်။ ဘာဖြစ်လို့လဲဆိုရင် သူက elements တွေကို normal document flow ရဲ့အပြင်ဘက်ကနေ ယူလိုက်လို့ဖြစ်ပါတယ်။ ပြီးတော့ ကျွန်တော်တို့မှာ clear fixes (သို့) vertical alignment issues တွေကိုလည်း စိုးရိမ်ရပါသေးတယ်။ ဒါကြောင့် new flex box feature ဟာဆိုရင် layout အတွက် တစ်ကယ်ကိုအသုံးဝင်ပြီး powerful ဖြစ်တဲ့ အခြေခံတစ်ခုဖြစ်ပါတယ်။ ပြီးတော့ layout အများစုမှာဖြစ်တဲ့ issues တွေကိုလည်း ဖြေရှင်းပေးပါတယ်။ အခုလက်ရှိမှာဆိုရင် new CSS Flex box feature ကို Safari, Firefox နဲ့ IE-- တွေမှာ support ပေးပါတယ်။ ကျွန်တော်တို့ရဲ့ latest specification နဲ့ syntax တွေကို support ပေးနိုင်တဲ့ browsers တွေကတော့ Chrome နဲ့ Opera တို့ပဲဖြစ်ပြီး မကြာခင် modern browser အားလုံးမှာ support ပေးနိုင်ဖို့ ရှိပါတယ်။ အခု Flex box တွေဘယ်လိုအလုပ်လုပ်လဲဆိုတာ ကြည့်ရအောင်။ Flex Box ဟာ 1-dimensional CSS layout ကို ကောင်းစွာ handle လုပ်နိုင်ပါတယ်။ ဘာဖြစ်လိုလဲဆိုရင် ကျွန်တော်တို့ဟာ elements တွေကို up, down, left, or right ဆိုပြီး ဘယ် direction မဆို layout ချနိုင်လို့ပဲဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ box တွေကို grow, shrink နဲ့ autofill space တွေကိုလည်း ပြုလုပ်နိုင်ပါတယ်။ ဒါကြောင့် line တစ်ခုထဲက boxes တွေပါတဲ့ အရာအားလုံးကို flex box တွေက ကောင်းစွာအလုပ်လုပ်ပေးနိုင်ပါတယ်။ ဒါကြောင့် ကျွန်တော်တို့ example ထဲက unordered list ထဲမှာ list item လေးခုရှိပြီး အဲ့ဒီ list itemလေးခုရဲ့ parent element ဖြစ်တဲ့ ul ကိုယူပြီး CSS file ထဲမှာ display: -webkit-flex; လို့ရေးပေးရပါမယ်။ အခုဆိုရင် Flex value ဟာ ကျွန်တော်တို့ရဲ့ unordered list ကို Flex Container Box တစ်ခုဖြစ်စေပြီး ul ရဲ့ content or children တွေကလည်း flex items အဖြစ်ပြောင်းလဲသွားပါတယ်။ အခုဆိုရင် browser မှာ ကျွန်တော်တို့ list items တွေက row တစ်ခုအတွင်း flex items တွေအဖြစ် ရှိနေမှာပါ။ ဒါပေမယ့် ကျွန်တော်တို့က 1 column အဖြစ် အဲ့ဒီ flex items တွေကို လိုချင်တယ်ဆိုရင် flex-flow ဆိုတဲ့ property ကိုအသုံးပြုရမှာဖြစ်ပြီး သူ့ရဲ့ value ကတော့ column ပဲဖြစ်ပါတယ်။ ကျွန်တော်တို့ column တွေကို ပြောင်းပြန်လုပ်ချင်ရင်တော့ column-reverse ကိုသုံးပေးရမှာဖြစ်ပါတယ်။ နောက် property တစ်ခုကတော့ Flex grow ဖြစ်ပြီး သူဟာ container ထဲမှာရှိတဲ့ ကျန်နေတဲ့ flex items တွေနဲ့ relative ဖြစ်စွာ grow ဖြစ်မယ့် flex item ဘယ်လောက်များလဲဆိုတာကို ဆုံးဖြတ်ပေးပါတယ်။ အခု ကျွန်တော်တို့ flex items တွေရဲ့ order ကို ပြောင်းလဲချင်တယ်ဆိုရင်တော့ order property ကိုအသုံးပြုရမှာဖြစ်ပြီး သူ့ရဲ့ value ကို -1 ပေးလိုက်မယ်ဆိုရင် ကျွန်တော်တို့ select လုပ်ထားတဲ့ item က list ရဲ့ရှေ့ဆုံးကိုရောက်သွားမှာဖြစ်ပါတယ်။ Flex box နဲ့ boxes တွေကို vertically align လုပ်ရတာပိုလွယ်ကူပါတယ်။ ပုံမှန်အားဖြင့်ဆိုရင် ကျွန်တော်တို့က margins, padding, line-height, or display တွေကိုသုံးပြီး လိုချင်တဲ့ vertical alignment တွေရဖို့ ပြုလုပ်နေရပါတယ်။ ဒါပေမယ့် flex box ကတော့ align-self ဆိုတဲ့ property ကိုသုံးပြီး vertical alignment ကို ဖန်တီးလို့ရပါတယ်။ align-self property မှာ value အနေနဲ့ center, flex-end, flex-start စသဖြင့်ရှိပါတယ်။ သူတို့အားလုံးဟာ flex container ထဲမှာတော့ wrap ဖြစ်နေမှာမဟုတ်ပါဘူး။ ပြောချင်တာက list items တွေက flex container ကိုကျော်ပြီး overflowing ဖြစ်နေပါတယ်။ ဒါကြောင့် ကျွန်တော်တို့ရဲ့ ul ထဲမှာ flex-wrap ဆိုတဲ့ property ကို value ဖြစ်တဲ့ wrap ဆိုပြီးထည့်ပေးဖို့လိုပါတယ်။

Multi Column Layout

နောက်ထပ် layout feature အသစ်တစ်ခုကတော့ multi-column layout ပဲဖြစ်ပါတယ်။ Multi-column layout ဟာ layout flexibility တွေပေးနိုင်တဲ့ Flex Box တွေနဲ့ဆင်တူပါတယ်။ ဒါပေမယ့် သူတို့နှစ်ခုရဲ့ ခြားနားချက်ကတော့ သူ့ကို multiple text columns တွေ create လုပ်ဖို့ text layout အတွက် design ပြုလုပ်ထားတာဖြစ်ပြီး သူ့ဟာ document အတွင်းမှာတော့ new box တစ်ခုလုံးနဲ့ new section တစ်ခုကိုတော့ create မလုပ်ပေးပါဘူး။ သူဟာ same box အတွင်းမှာပဲ box ရဲ့ content ကို multiple columns တွေအဖြစ် flowing လုပ်ခြင်းဖြင့် block layout mode ကို extend လုပ်ပေးရုံပဲဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ columns ရဲ့ widths, gaps, rules နဲ့ နောက်ထပ် simple CSS properties အများကြီးကို determine လုပ်ပေးနိုင်ပါတယ်။ Multi-column layout ဟာ content က multiple column boxes တွေထဲကို flow ဖြစ်နေတဲ့ column box container ရဲ့ new type တစ်ခုနဲ့ မိတ်ဆက်ပေးလိုက်ပါတယ်။ အောက်က example မှာဆက်ကြည့်ရအောင်။

CSS Blend Modes

နောက်ထပ် feature တစ်ခုကတော့ ကျွန်တော်တို့ရဲ့ graphics editor ထဲက tool တစ်ခုနဲ့ ဆင်တူဖြစ်တဲ့ blend modes ပဲဖြစ်ပါတယ်။ CSS blend ဆိုတာ two or more layers တွေကို အတူတကွ blended လုပ်ထားပြီး သူတို့ဟာ exciting ဖြစ်တဲ့ visual effects တွေကို create လုပ်ဖို့ ခွင့်ပြုပေးပါတယ်။ အဲ့ဒီ blending effects တွေကို graphic design or digital image editing တွေမှာ အသုံးပြုနိုင်ပါတယ်။ ပြီးတော့ ကျွန်တော်တို့ဟာ browser ထဲမှာရှိတဲ့ အဲ့ဒီ same blending effects တွေကို CSS blend modes တွေကိုသုံးပြီး recreate လုပ်နိုင်ပါတယ်။ Blend modes တွေဟာ compositing နဲ့ blending CSS specification တွေရဲ့ အစိတ်အပိုင်းတစ်ခုပဲဖြစ်ပါတယ်။ ဒါဆို compositing and blending ဆိုတာ ဘာလဲကြည့်ရအောင်။ Compositing ဆိုတာ element တစ်ခုကို သူ့ရဲ့ backdrop နဲ့ ဘာကို combine လုပ်မလဲလို့ ဆိုလိုတာဖြစ်ပြီး back drop ဆိုတာကတော့ element ရဲ့နောက်မှာရှိတဲ့ content ပဲဖြစ်ပါတယ်။ Blending ဆိုတာကတော့ element တစ်ခုကဘယ်နေရာမှာပဲဖြစ်ဖြစ်၊ backdrop တစ်ခုက ဘယ်နေရာမှာပဲ overlap ဖြစ်ဖြစ် အဲ့ဒီဖြစ်တဲ့နေရာက colors တွေရဲ့ mixing
