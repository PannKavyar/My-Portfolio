Introduction to JavaScript

JavaScript ဟာ popular အဖြစ်ဆုံးနဲ့ ဘက်စုံသုံးလို့ရတဲ့ programming language တစ်ခုဖြစ်ပါတယ်။ သူ့ကို browser မှာ built လုပ်ရတာဖြစ်တဲ့အတွက် quick & easy ဖြစ်ပါတယ်။ သူမှာ simple straight forward syntax တစ်ခုရှိပါတယ်။ JavaScript မှာ growing vibrant support community တစ်ခုရှိပြီး အဲ့ဒီ community ကပဲ ကျွန်တော်တို့ကို အလုပ်မြန်မြန်ပြီးမြောက်စေဖို့ lots of tools တွေကို built လုပ်ပေးနေပါတယ်။ တစ်ကယ်လို့ ကျွန်တော်တို့အနေနဲ့ အခြား language တစ်ခုကနေ JavaScript ကိုပြောင်းပြီးလေ့လာမယ်ဆိုရင်လည်း JavaScript ဟာ အခြား programming languages တွေရဲ့ core concepts တွေနဲ့ capabilities တွေကို share လုပ်ပေးပါတယ်။ ဒါကြောင့် JavaScript ဟာဆိုရင် 2018 ခုနှစ်နောက်ပိုင်းမှာ ဝင်ငွေအကောင်းဆုံး programming language တစ်ခုဖြစ်ခဲ့ပါတယ်။

Simple Command in JavaScript

JavaScript ကို စပြီးလေ့လာတော့မယ်ဆိုရင် ကျွန်တော်တို့အရင်ဆုံးသိထားရမှာကတော့ JavaScript development မှာရှိတဲ့ common data types, variables, arrays, objects, loops, function, နဲ့ အချို့သောအရာတွေပဲဖြစ်ပါတယ်။ JavaScript programming မှာ JavaScript ကို encounter လုပ်ဖို့ အလွယ်ကူဆုံးနေရာတစ်ခုကတော့ web browser တစ်ခုပဲဖြစ်ပါတယ်။ Browsers တွေဟာ JavaScript program တစ်ခုအတွင်းမှာရှိတဲ့ instruction တွေကို read, understand နဲ့ run လုပ်ဖို့အတွက် built-in JavaScript interpreter တစ်ခုရှိပါတယ်။ JavaScript ကိုစရေးမယ်ဆိုရင် အရင်ဆုံး browser တစ်ခုနဲ့ သူ့ထဲက built-in ပါတဲ့ console တစ်ခုရယ် လိုအပ်ပါတယ်။ အဲ့ဒီ console tab အတွင်းမှာ JavaScript code တွေကို run နိုင်ပါတယ်။ JavaScript ကိုသုံးရခြင်းအကြောင်းက information data လို့ခေါ်တဲ့ ပေးထားတဲ့ information တွေနဲ့အလုပ်လုပ်ဖို့ပဲဖြစ်ပါတယ်။ JavaScript ဟာ data types အမျိုးမျိုးကို support လုပ်ပေးပါတယ်။ ဥပမာ 20 ဆိုတဲ့ number ဟာ numeric data type တစ်ခုဖြစ်ပြီး အဲ့ဒီ numbers တွေကို JavaScript မှာ adding, subtracting, computing total costs, game score စတဲ့ calculations တွေတွက်ထုတ်ဖို့အတွက် အသုံးပြုပါတယ်။ Numbers တွေကလည်း whole numbers (သို့) numbers with decimal points တွေနဲ့ ဖြစ်နိုင်ပါတယ်။ JavaScript ဟာ addition (+), subtraction (-), multiplication (*) နဲ့ division (/) ဆိုတဲ့ arithmetic operators တွေကိုလည်း provide လုပ်ပေးထားပါတယ်။ နောက်ထပ် JavaScript မှာ comm ဖြစ်တဲ့ data type တစ်ခုကတော့ string ပဲဖြစ်ပါတယ်။ Strings တွေကို ကျွန်တော်တို့ရဲ့ program မှာ words, sentences နဲ့ အခြား text တွေထည့်ဖို့အတွက် သုံးပါတယ်။ ကျွန်တော်တို့ဟာ string တစ်ခုကို series of letters, numbers နဲ့ other characters တွေကို single or double quotation marks အတွင်းမှာ ရေးလေ့ရှိပါတယ်။ Alert() ဆိုတာ browser မှာ pop up & dialog box တစ်ခုပြပေးဖို့အတွက် အသုံးပြုတဲ့ browser specific function တစ်ခုဖြစ်ပါတယ်။ Console ထဲမှာ ရေးတဲ့ code ဟာ JavaScript ရဲ့ statement ဖြစ်ပြီး JavaScript မှာ statement တစ်ခုပြီးတိုင်း semi-comma ထည့်ပေးရပါမယ်။ Alert ဟာ message တစ်ခုကို display ပြတဲ့ dialog box တစ်ခုသာဖြစ်ပြီး browsers တွေဟာ site visitors တွေဆီက input ကို လှမ်းဖမ်းဖို့အတွက် prompt ဆိုတဲ့ simple command တစ်ခုကို ပေးထားပါတယ်။ Prompt() ဟာ user ဆီကို question တစ်ခုတောင်းပြီး user ဆီက response တစ်ခုယူလိုက်တာဖြစ်ပါတယ်။ Browser က promt ကနေ response လုပ်လိုက်တာကို capture လုပ်ပြီး value ကို return ပြန်ပေးလိုက်တာဖြစ်ပါတယ်။

Variables in JavaScript

JavaScript ဟာ program တစ်ခုအတွင်းမှာရှိတဲ့ information တွေကို Variables တွေနဲ့ store & track လုပ်ပေးပါတယ်။ Variables တွေဟာ JavaScript မှာ မရှိမဖြစ်အရေးပါတဲ့ concepts တွေထဲက တစ်ခုဖြစ်ပါတယ်။ ဥပမာ ကျွန်တော်တို့က message တစ်ခုကို dialogue box နဲ့ prompt လုပ်ပြချင်တယ်ဆိုရင် အဲ့ဒီ message ကို variable တစ်ခုထဲမှာ အရင်ထည့်ပြီးတော့မှ အသုံးပြုပါတယ်။ Variable တစ်ခုကို ကြေငြာမယ်ဆိုရင် var ဆိုတဲ့ keyword နဲ့ သူ့နောက်မှာ ကျွန်တော်တို့ပေးချင်တဲ့ variable name ကို ရေးပေးရပါတယ်။ နောက်ပြီးရင်တော့ အဲ့ဒီ variable ထဲကို value ထည့်ဖို့အတွက် assign လုပ်တာလို့ခေါ်တဲ့ equal (=) ရေးပေးရပါတယ်။ Equal ရဲ့နောက်မှာတော့ ကျွန်တော်တို့ variable ထဲကို assign လုပ်မယ့် value ထည့်ပေးရတာဖြစ်ပါတယ်။ ကျွန်တော်တို့ ကြေငြာထားတဲ့ variable တွေကို program ရဲ့ကြိုက်တဲ့နေရာကနေ ခေါ်သုံးနိုင်ပါတယ်။ အခုဆိုရင် JavaScript မှာ variable တွေကို let or const ဆိုတဲ့ keywords တွေနဲ့လည်း ကြေငြာလို့ရနေပါပြီ။ ဒါပေမယ့် ကျွန်တော်တို့သုံးတဲ့ declaring rules တွေအပေါ်မူတည်ပြီးတော့ သူတို့ရဲ့ values တွေက ပြောင်းလဲသွားပါလိမ့်မယ်။ JavaScript မှာ strings တွေတစ်ခုနဲ့တစ်ခု combining လုပ်ခြင်းကို concatenation လို့ခေါ်ပါတယ်။ ကျွန်တော်တို့ရဲ့ string နှစ်ခုကို concatenate လုပ်မယ်ဆိုရင်တော့ plus sign (+) ကို string နှစ်ခုကြားမှာ ထည့်ရေးပေးရပါတယ်။

Functions in JavaScript

Parameter in JavaScript

ကျွန်တော်တို့ဟာ message ကဲ့သို့သော information တွေကို prompt & alert ထဲကို pass လုပ်သလိုပဲ function တစ်ခုထဲကို information တွေ pass လုပ်ဖို့ဆိုရင် ကျွန်တော်တို့ function ရဲ့ parenthesis အတွင်းမှာမှာ parameter တစ်ခုကို ထည့်ပေးရမှာဖြစ်ပါတယ်။ ကျွန်တော်တို့ function ထဲက parameter တစ်ခုဟာ variable တစ်ခုလိုပဲ အလုပ်လုပ်ပေးပါတယ်။ အဲ့ဒီ parameter အလုပ်လုပ်ဖို့ဆိုရင်တော့ function call ခေါ်တဲ့အချိန်မှာ parameter value ကို parenthesis ထဲမှာ ထည့်ပေးရမှာဖြစ်ပါတယ်။ Function call ထဲကို information တွေ pass လုပ်ခြင်းဟာ function တစ်ခုထဲကို argument တစ်ခု pass လုပ်လိုက်တာဖြစ်ပါတယ်။ ဒါကြောင့် functions တွေထဲမှာ parameter တွေကို အသုံးပြုခြင်းဟာ ပိုပြီးတော့ flexible ဖြစ်စေပြီး values အမျိုးမျိုးကို pass လုပ်နိုင်သလို results အမျိုးမျိုးလည်း ထုတ်နိုင်မှာဖြစ်ပါတယ်။

Multiple Parameters in JavaScript

JavaScript မှာ arrays တွေနဲ့ နောက်ထပ် ယှဉ်တွဲပြီး အသုံးများတာတစ်ခုကတော့ loops တွေပဲဖြစ်ပါတယ်။ Looping ဟာ programming မှာ တစ်ကယ်ကို powerful ဖြစ်တဲ့ tool တစ်ခုဖြစ်ပါတယ်။ တူညီတဲ့ actions တွေကိုပဲ ထပ်ခါထပ်ခါ အလုပ်လုပ်စေချင်တဲ့အတွက် loops တွေကိုသုံးတာဖြစ်ပါတယ်။ JavaScript မှာလည်း loop အမျိုးမျိုးရှိပါတယ်။ အခုကျွန်တော်တို့ဟာ JavaScript မှာ အသုံးများတဲ့ loop နှစ်မျိုးကို လေ့လာသွားပါမယ်။ ပထမတစ်ခုကတော့ foreach() loop ဖြစ်ပြီး foreach loop ဟာ array တစ်ခုထဲက item တစ်ခုချင်းစီကို access လုပ်ပေးတာပဲဖြစ်ပါတယ်။ Foreach loop ကိုရေးမယ်ဆိုရင်တော့ forEach ဆိုတဲ့ key နဲ့စရေးရပြီး သူ့နောက်က parenthesis ထဲမှာတော့ array ထဲက item တစ်ခုချင်းစီကို ထုတ်ဖို့အတွက် callback function တစ်ခုရေးရမှာဖြစ်ပါတယ်။ အောက်က ဥပမာမှာဆိုရင် function ထဲမှာ movie ဆိုတဲ့ parameter တစ်ခုလက်ခံထားပြီး အဲ့ဒီ movie ဟာ current element or array ထဲမှာ process လုပ်မယ့် movie title ပဲဖြစ်ပါတယ်။ အဲ့ဒီမှာဆိုရင် function ရဲ့နောက်မှာ name လိုက်ပြီးမရေးထားပါဘူး။ သူ့ကို anonymous function လို့ခေါ်ပါတယ်။ ကျွန်တော်တို့အနေနဲ့ anonymous functions တွေကနေ အခြား function ကို parameters အဖြစ်လှမ်းပို့ချင်တဲ့အခါမှာ anonymous function ကို အသုံးပြုလေ့ရှိပါတယ်။ Foreach ရဲ့သဘောတရားကတော့ သူ့ထဲကို pass လုပ်မယ့် function ကို ခေါ်ပေးတာဖြစ်ပါတယ်။ အဲ့ဒီမှာ function ကိုခေါ်လိုက်ပြီဆိုတာနဲ့ လက်ရှိပထမဆုံး movie title က movie ဆိုတဲ့ argument ထဲကို pass လုပ်သွားပြီး နောက်တစ်ခု ၊ နောက်တစ်ခု နဲ့ array ထဲက movie title တွေကုန်တဲ့အထိ loop ပတ်သွားတာဖြစ်ပါတယ်။

ဒီတစ်ခါမှာတော့ ကျွန်တော်တို့က movie object ထဲက status property ရဲ့ value အပေါ်မူတည်ပြီးတော့ message တစ်ခုကို display ပြပေးမယ့် shorthand program တစ်ခုကို ရေးသွားမှာဖြစ်ပါတယ်။ တစ်ကယ်လို့ status က Available နဲ့ ညီရင် movie title နဲ့ showtime ကို console မှာ display လုပ်သွားပါမယ်။ တစ်ကယ်လို့ status က Unavailable နဲ့ညီနေရင်တော့ movie က sold out ဖြစ်သွားပြီဆိုတဲ့ message ကိုပြပေးမှာဖြစ်ပါတယ်။ အဲ့ဒီအတွက် ကျွန်တော်တို့မှာ movies array ထဲက object တစ်ခုချင်းစီကို လိုက်စစ်ဖို့အတွက် loop တစ်ခုလိုအပ်ပါတယ်။ ကျွန်တော်တို့ဟာ same actions တွေကိုပဲ repeat လုပ်နေမှာဖြစ်တဲ့အတွက် for loop ကိုအသုံးပြုပါမယ်။ အဲ့ဒီမှာဆိုရင် plus sign(+) ကို strings တွေ combine လုပ်ဖို့အတွက် သုံးထားတာဖြစ်ပါတယ်။ နောက်ပြီးတော့ strict equality operator (===) လို့ခေါ်တဲ့ triple equal sign ဟာ values နှစ်ခုအတိအကျ ညီရမယ်လို့ စစ်တာဖြစ်ပါတယ်။ အောက်က example ကိုကြည့်ရအောင်။

Using Backtick Character

ဒီတစ်ခါမှာတော့ ကျွန်တော်တို့က string တွေကို combine လုပ်ပေးတဲ့ quotes တွေနဲ့ plus sign (+) တွေကိုမသုံးဘဲ template literal တစ်ခုကို အသုံးပြုပါမယ်။Template literal တစ်ခု complete ဖြစ်ဖို့ Single or double quotes အစား backtick ဆိုတဲ့ character ကိုအသုံးပြုရပါမယ်။ Value တစ်ခုကို insert လုပ်ဖို့ plus sign ကိုအသုံးပြုမယ့်အစား string ထဲကို value ထည့်ဖို့အတွက် ${} ဆိုတဲ့ syntax ကိုအသုံးပြုပါမယ်။ ဒီလိုအသုံးပြုခြင်းဟာ code ကို ရေးရလွယ်ကူသလို ဖတ်ရလို့လည်း လွယ်ကူပါတယ်။

Example of HTML File:

        
            <!DOCTYPE html>
            <html lang="en">
            
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Using Backtick Character</title>
            </head>
            
            <body>
            
                <h1>My Web Page</h1>
            
                <script src="../js/backtick_char.js"></script>
            </body>
            
            </html>

Example of JS File:

        
            var movies = [{
                title: 'Peaky Blinders',
                time: '12pm',
                status: 'Available'
            },
            {
                title: 'Avengers',
                time: '3pm',
                status: 'Unavailable'
            },
            {
                title: 'The Last Jedi',
                time: '6pm',
                status: 'Available'
            },
            {
                title: 'Lady Bird',
                time: '8pm',
                status: 'Unavailable'
            }
        ];
        
        for (var movie of movies) {
            if (movie.status === 'Available') {
                //using backtick character
                console.log(`The movie ${movie.title} plays at ${movie.time}.`);
            } else {
                console.log(`The movie ${movie.title} is sold out.`);
            }
        }
